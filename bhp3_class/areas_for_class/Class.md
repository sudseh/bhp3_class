## TCP Client

```python
import socket

def connect(host, port):
    client = socket.socket(socket.AF_INET, socket.SOCK_STREAM)
    client.connect((host, port))
    client.send('GET /')
    return client.recv(4096)

if __name__ == '__main__':
    host = '192.168.1.69'
    port = 9999
    print(connect(host, port))
```
